甲板部

船长：全船的管理与驾驶

大副：甲板部分的工作、货物的配载、装卸和运输管理

二副：驾驶任务，驾驶设备的技术管理，航海图书资料管理

三副：船舶航行、停泊、主管救生、消防设备的技术管理

水手长：带领水手进行甲板的日常维护保养工作

水手：跟随高级船员负责正常的航行值班。

轮机部

轮机长：船舶机械推进职能

大管轮：轮机部设备的安全和预防

二管轮：辅机电气设备管理。

三管轮：辅机电气设备管理。

机工：跟随高级船员负责正常的航行值班。

电机员：主管船舶电机和船上电气设备
